# GitHub Pages

GitHub Pages 是 GitHub 提供的一個免費的**靜態網站託管服務**，它允許使用者在 GitHub 上託管靜態網站，而不需要使用額外的網站託管服務。GitHub Pages 可以用來建立個人網站、部落格、作品集等，它非常簡單易用，而且可以與 GitHub 的其他功能很好地整合。

GitHub Pages 支持多種語言和框架，包括 **HTML、CSS、JavaScript、Jekyll 和 Hugo**。您可以使用任意語言或框架來建立您的網站。

以下是一些使用 GitHub Pages 的優點：

- **免費**
- 簡單易用
- 與 GitHub 的其他功能很好地整合
- 支持多種語言和框架

以下是一些使用 GitHub Pages 的缺點：

- **只能託管靜態網站**
- 無法使用動態功能
- 無法使用複雜的網站設計
