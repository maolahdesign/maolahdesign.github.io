# AI應用規劃師-智能報表應用 (12H)

## 課程介紹

結合人工智慧與 Excel 實務操作，從基礎函數到進階資料處理，帶你掌握數據分析與視覺化技巧。課程深入介紹生成式 AI 應用與 Excel 智能整合，提升工作效率與決策力。適合希望善用 AI 優化日常辦公與數據管理的專業人士。

#### 1.2 常用函數介紹

##### 運算函數
- **SUM, SUMIF, SUMIFS** - 數值加總與條件加總
- **ROUND, ROUNDUP, ROUNDDOWN** - 數值四捨五入處理
- **ABS, MOD** - 絕對值與餘數計算
- **MAX, MIN, AVERAGE** - 最大值、最小值與平均值

##### 日期與時間函數
- **YEAR, MONTH, DAY** - 日期元件擷取
- **TODAY, NOW** - 當前日期與時間
- **DATEVALUE, TIMEVALUE** - 日期時間轉換
- **WEEKDAY, NETWORKDAYS** - 工作日計算

##### 邏輯函數
- **IF, IFS** - 條件判斷與多重條件
- **AND, OR, NOT** - 邏輯運算子
- **IFERROR, IFBLANK** - 錯誤處理函數

**章節說明：**
本節深入學習最常用的 Excel 函數，透過實際案例練習，讓學員熟練掌握各類函數的應用場景和技巧。

## ⚡ 課程實作建議

### 📋 學習策略
1. **循序漸進練習** - 每個章節完成後立即練習相關案例
2. **建立個人資料庫** - 收集各類型的練習資料集
3. **記錄學習筆記** - 整理常用函數和技巧的個人手冊
4. **參與社群討論** - 加入 Excel 相關社群分享學習心得

### 🛠 實作專案建議
1. **個人財務分析儀表板** - 整合收支數據的動態分析報告
2. **銷售績效追蹤系統** - 結合樞紐分析表和圖表的管理工具
3. **庫存管理自動化系統** - 利用函數和 AI 建立智能補貨提醒
4. **客戶分析報告** - 從原始數據到商業洞察的完整流程

### 📊 評估與驗證
1. **定期技能檢測** - 使用課程提供的評估工具
2. **同儕互評** - 與同學交換作品進行相互評估
3. **實際應用驗證** - 將學到的技能應用到實際工作中
4. **持續更新學習** - 跟上 Excel 和 AI 工具的最新發展
