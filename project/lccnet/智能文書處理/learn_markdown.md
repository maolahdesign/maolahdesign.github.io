## Markdown 語法速成教學 (20 分鐘)

**課程目標：** 讓學員掌握基礎 Markdown 語法，並能利用簡單語法書寫、格式化文件。

**適合對象：** 想要快速學習 Markdown 語法的初學者。

**教學工具：** 網頁瀏覽器 (例如 Chrome, Firefox)，Markdown 編輯器 (可選，例如 Typora, Visual Studio Code)

---

**1. 什麼是 Markdown? (2 分鐘)**

* **定義:** Markdown 是一種輕量級的標記語言，用簡單易讀的文字符號來格式化文件。
* **優點:**
    * 易於學習和使用。
    * 平台獨立 (可以在任何文本編輯器中編寫)。
    * 易於轉換成 HTML、PDF 等格式。
    * 適合寫文件、筆記、部落格文章等。
* **用途:** 常見於 GitHub、GitLab、README 文件、部落格系統等。

**2. 基礎語法 (10 分鐘)**

* **標題 (Headers):** 使用 `#` 符號表示標題層級。
    * `# 標題 1`
    * `## 標題 2`
    * `### 標題 3`
    * …
* **強調 (Emphasis):** 使用 `*` 或 `_` 表示斜體，使用 `**` 或 `__` 表示粗體。
    * `*斜體*` 或 `_斜體_`
    * `**粗體**` 或 `__粗體__`
* **列表 (Lists):** 使用 `-`, `+` 或 `*` 表示無序列表，使用數字加上 `.` 表示有序列表。
    * 無序列表:
        `- 項目 1`
        `- 項目 2`
        `- 項目 3`
    * 有序列表:
        `1. 項目 1`
        `2. 項目 2`
        `3. 項目 3`
* **連結 (Links):** 使用 `[連結文字](連結網址)` 格式。
    * `[Google](https://www.google.com)`
* **圖片 (Images):** 使用 `![圖片描述](圖片網址)` 格式。
    * `![Markdown Logo](https://markdown-here.com/img/icon256.png)`
* **程式碼 (Code):** 使用 `` ` `` 符號包裹程式碼。
    * `int main() {  printf("Hello, world!\n"); }`
* **引用 (Blockquotes):** 使用 `>` 符號表示引用。
    * `> 這是引用內容`
* **水平線 (Horizontal Rule):** 使用 `---`, `***` 或 `___` 表示水平線。

**3. 進階語法 (5 分鐘)**

* **表格 (Tables):** 使用 `|` 和 `-` 表示表格。

    | Header 1 |  | Header 2 |Header 2 |
    | --- | --- |--- |--- |
    | Cell 1   |    | Cell 2   |Cell 2   |
    |  |  |  | |
    | hello |  | hey|hey|

    ```markdown
    
    
    
    ```

* **任務列表 (Task Lists):** 使用 `- [ ]` 表示未完成任務，使用 `- [x]` 表示已完成任務。
    
    * `- [ ] 待辦事項 1`
    * `- [x] 已完成事項 1`
    
* **換行 (Line Breaks):** 在行尾加上兩個或多個空格才能換行。

**4. 實作練習 (3 分鐘)**

* **練習題目:**  請用 Markdown 語法撰寫一篇關於 "Markdown 入門" 的短文，包含標題、粗體、列表、連結和圖片。
* **示範:**  講師現場示範如何用 Markdown 語法撰寫短文。
